## 🚀 Installation & Setup
### Prerequisites
* Python 3.8 or newer installed on your system.
* Git installed.
### Windows
**Clone the repository:**
```cmd
git clone https://github.com/admJvsc/projekt-zabka-sklep.git
```
```cmd
cd projekt-zabka-sklep
```
**Create and activate a virtual environment:**
```cmd
python -m venv venv
```
```cmd
venv\Scripts\activate
```
**Install the package and dependencies:**
```cmd
pip install .
```
**Run the application:**
```cmd
python -m frog_package.gui
```

### Linux
**Clone the repository:**
```bash
git clone https://github.com/admJvsc/projekt-zabka-sklep.git
```
```bash
cd projekt-zabka-sklep
```
**Create and activate a virtual environment:**
```bash
python3 -m venv venv
```
```bash
source venv/bin/activate
```
**Install the package and dependencies:**
```bash
pip install .
```
**Run the application:**
```bash
python3 -m frog_package.gui
```
