## 🚀 Instalacja i uruchomienie
### Wymagania wstępne
* Python 3.8
* Git
### Windows
**Sklonuj repozytorium:**
```cmd
git clone https://github.com/admJvsc/projekt-zabka-sklep.git
```
```cmd
cd projekt-zabka-sklep
```
**Utwórz i uruchom środowisko wirtualne:**
```cmd
python -m venv venv
```
```cmd
venv\Scripts\activate
```
**Zainstaluj pakiet i zależności:**
```cmd
pip install .
```
**Uruchom aplikację:**
```cmd
python -m frog_package.gui
```

### Linux
**Sklonuj repozytorium:**
```bash
git clone https://github.com/admJvsc/projekt-zabka-sklep.git
```
```bash
cd projekt-zabka-sklep
```
**Utwórz i uruchom środowisko wirtualne:**
```bash
python3 -m venv venv
```
```bash
source venv/bin/activate
```
**Zainstaluj pakiet i zależności:**
```bash
pip install .
```
**Uruchom aplikację:**
```bash
python3 -m frog_package.gui
```
